# Roadmap

## Phase 1 – Core
- Workspace-based structure
- Auth (invite-only, email/password)
- Workspace and membership CRUD
- Project & Task CRUD
- Realtime task updates
- Role-based access control
- Task dependencies
- Toast/in-app alerts

## Phase 2 – Collaboration
- Chat (per-workspace or per-project channels)
- Comments on tasks/projects
- File uploads (avatars, attachments)
- Activity log UI
- Mentions (with automatic notifications)

## Phase 3 – Polish
- Notifications panel (in-app + toast)
- Global search (initial full-text)
- UI themes
- Filters by assignee, status, due date
